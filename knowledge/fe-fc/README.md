# knowledge/fe-fc — FC版FE 知識ファイル（攻略データではない）

AIの**感情・反応を正しくする前提**と、**今この章に誰がいるか**だけを持つ最小限の知識。2階建て構成（`REQUIREMENTS §8`）。

- `fe-primer.md` … 全章共通プライマー1枚。システムプロンプト先頭に固定。
- `chapters/chapter-XX.md` … 章ごとのキャスト表。`state.chapter` に対応する**1枚だけ**注入する（番号で引くだけの最小リトリーバル）。`chapter-01.md`, `chapter-02.md` … と**ASCII・ゼロ埋め2桁**で並べる。日本語の章タイトルは中の見出し（`# 第1章 …`）で保持。

ローダーは `src/lib/knowledge.ts`（`loadPrimer` / `loadChapterCast`）。`state.chapter` が「第3章」なら `chapters/chapter-03.md` を読む。該当ファイルが無ければ未注入で続行する。

## 章キャスト表の作り方（スクショ → 名簿化・§8.4）

章を進めるたびに、その章の登場人物一覧の**スクリーンショットを撮って Claude Code に渡し**、テキスト名簿に起こす。指示には必ず次の3点を添える。

1. **§8.2 の体裁テンプレと同じ形式**で `chapters/chapter-XX.md` を作る（下記テンプレ）。
2. **スクショの表の情報だけ**をテキスト化する。**攻略・解説・ストーリー・先の展開は書かない**（ネタバレ禁止・§5.2）。
3. ファイル名は **ASCII・ゼロ埋め番号**。日本語タイトルは中の見出しに。

### 体裁テンプレ（§8.2・統一する）

```markdown
# 第1章 マルスの旅立ち

## 自軍（仲間）
- マルス／ロード／Lv1 HP18／持ち物: レイピア
- ジェイガン／パラディン／Lv1 HP20／持ち物: 鉄の剣・銀の槍

## 敵
- ガザック／かいぞく／Lv5 HP24／持ち物: 鋼の斧（このマップのボス）
- ガルダへい（×11）／かいぞく／Lv1 HP18／持ち物: 斧
```

## 守ること

- **ネタバレの線引き**：「この章で登場/仲間になる人」まで。「この後死ぬ／裏切る」等の先の運命は書かない（一緒に驚くため）。
- **マップ画像は含めない**（戦友の役割に不要・Phase 2 で必要になれば検討）。
- **攻略サイトの本文・画像は転載しない**。誰がいるか＝事実を自分の言葉のテキスト名簿にする。
- 読み取れない値は空欄か省略。**表に無い項目を創作・補完しない**。
- 不確かな仕様は `⚠️要確認` を付け、**版の取り違えチェックは開発者が一次情報で行う**（FC版＝FE1・1990）。
