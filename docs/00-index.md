## 進め方

- MVP の最短経路は **01→02→03→04→05→06→07→08→10**。09・11 は UX 仕上げ、12・13 は任意。
- Phase 2 は **14→15→16** の順（17 は独立・いつでも可）。14 だけで体感が変わるので、一度動かして話し方を確かめてから 15・16 を積む。
- 先読み実装をしない。`REQUIREMENTS.md` に無い機能は、まずチケットを起こす。
- 「やらないこと」の強い制約は **14 で一部が変わる**（ネタバレ禁止は撤廃、攻略アドバイス禁止は維持）。14 完了後は `REQUIREMENTS.md §5.2 / §12` を正とする。
